# Mastering RabbitMQ
## 开始
### 使用场景：
- 确保消息被传递和接受
- 将消息路由到正确的目的地
- 保存消息的状态
- 支持多种传输协议（AMQP, MQTT, STOMP, HTTP)
- 支持集群
- 高可用，可扩展，可伸缩。
- 支持插件
- 支持多种语言客户端

### 安装与启动
sudo apt-get install rabbitmq-server && sudo rabbitmq-server
